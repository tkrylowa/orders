Реализовать метод сервиса Mono<Order> makeOrder(Map<Integer, Integer>),
 который принимает на вход мапу, где ключи - идентификаторы товаров,
 значения - количество товаров
 и возвращает возвращает результат в виде Mono<Order>

Order
{
  products: [
    {
      product_id: 1,
      count: 2
    }
  ],
  price: общая сумма за заказ

Данные о товарах получаем от внешней системы
по идентификаторам товаров
GET /api/v1/catalog/ids=34,89,22,134

Пример ответа:
[
  {
    id: 34,
    price: 100
  }
]
